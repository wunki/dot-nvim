# Dot Neovim

This is my minimal Neovim configuration.

## Key bindings

| Key                 | Description                        |
|---------------------|------------------------------------|
| <kbd>Space</kbd> <kbd>ff</kbd> | Find files in CWD                |
| <kbd>Space</kbd> <kbd>fd</kbd> | Open files in dotfiles directory |
| <kbd>Space</kbd> <kbd>fc</kbd> | Open Neovim config files         |
