# Dot Neovim

This is my minimal Neovim configuration.

## Key bindings

| Key              | Description                        |
|------------------|------------------------------------|
| <kbd><space>ff</kbd> | Find files in CWD                |
| <kbd><space>fd</kbd> | Open files in dotfiles directory |
| <kbd><space>fc</kbd> | Open Neovim config files         |
